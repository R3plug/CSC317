Projects

Automating device restarts via database collation between Jamf and Zscaler APIs

Using the APIs for Jamf and Zscaler I pull active devices from both portals to json files. Then from the zscaler
list there is a field that only refreshes if the device is restarted. I compare this time to the local system time
and remove any devices from the list that have been restarted in the last week. Between the two lists there is no data that matches
so I edit the user info in both lists to create a standard user name to match up the data. Then I can find the Jamf devices
that need to be restarted and trigger the restarts via the Jamf API.
